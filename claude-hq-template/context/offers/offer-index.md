# Offers

> Template. One file per offer. Your Cofounder helps you add these as your
> offers take shape. This index is a quick reference summary of all of them.

## How to use this folder

- `offer-index.md` (this file): one-line summary of every offer.
- `offer-[name].md`: one file per offer with the full details (promise, price,
  who it's for, what's included, the outcome it delivers).
